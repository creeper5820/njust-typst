# njust/example

这是一个匿名化的本科毕业论文示例，用于演示 `njust` 模板的完整使用方式。

特点：

- 结构完整，包含封面、声明、摘要、目录、图表目录、正文、致谢、参考文献与附录。
- 内容为示例文本，不包含真实人名、真实项目经历或真实单位信息。
- 可直接复制后替换题目、作者、章节与参考文献内容。
- 示例正文已包含引用，因此默认编译结果中的参考文献页不是空白页。

## 编译方式

由于 `main.typ` 通过 `../lib.typ` 引用模板，需将 Typst 项目根设置到 `njust/` 目录：

```bash
cd njust/example
typst compile --root .. main.typ
```

若需要加载模板字体，可同时设置 `TYPST_FONT_PATHS`。
